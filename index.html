My Process

 
 
 
 
 

Conversation

Conversation with the client about the requirements, goals, scope, assumptions and timeline of the project and technologies to be implemented

 
 
 
 
 

Planning

Create a detailed plan that contains the steps of the development and a date estimation for completion

 
 
 
 
 

Implementation/Coding Solution

Start development with full visibility from the client of each tasks in progress/completed implementing agile methodology if possible, and trying to get a daily scrum or another preferred method in order to keep track of the project

 
 
 
 
 

QA Review / Feedback

Review of the project and each task need to be ensured that it's working as expected

 
 
 
 
 

Delivery

Deploy/Delivery code to client or production environment
